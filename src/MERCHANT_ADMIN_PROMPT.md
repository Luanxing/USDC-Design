# 商户管理后台 - 支付管理系统提示词

## 项目概述
为"カフェMax"餐厅的Web3支付系统创建一个桌面端商户管理后台，专注于支付交易管理、收款配置和数据分析功能。采用现代化的日式金融科技美学，延续客户端的设计语言。

---

## 设计规范

### 视觉风格
- **主色调**: 薄荷水绿色 (#00C2A8) - 用于关键操作和成功状态
- **强调色**: 币安黄 (#FFD34E) - 用于重要提示和警告
- **背景色**: 
  - 主背景: #F7F9FB
  - 卡片背景: #FFFFFF
  - 次要背景: #F8FAFC
- **文字颜色**:
  - 主文字: #1E293B (slate-900)
  - 次要文字: #64748B (slate-500)
  - 占位文字: #94A3B8 (slate-400)

### 字体
- **英文**: SF Pro Display / Inter
- **日文**: Noto Sans JP
- **中文**: Noto Sans SC

### 布局
- **响应式**: 桌面优先（1440px - 1920px），支持平板横屏
- **侧边栏宽度**: 280px
- **内容区最大宽度**: 1200px
- **卡片圆角**: 16px
- **按钮圆角**: 12px

## 核心功能模块

### 1. 仪表盘 (Dashboard)

#### 顶部概览卡片（4个）
```
┌────────────────────────────────────────────────────────��┐
│  📊 今日收入             💳 交易笔数                    │
│  ¥245,680               158笔                          │
│  📈 +12.5% vs 昨日      📈 +8 笔                       │
├─────────────────────────────────────────────────────────┤
│  ⚡ 成功率               ⏱️ 平均处理时间                │
│  98.2%                  2.4秒                          │
│  ✅ 正常                📉 -0.3秒                      │
└─────────────────────────────────────────────────────────┘
```

**数据要求**:
- 实时更新（每30秒）
- 同比数据显示
- 状态指示器（绿色=良好，黄色=警告，红色=异常）

#### 实时交易流（Live Feed）
- **显示内容**:
  - 订单号（#ORD-20251104-001）
  - 支付方式图标（USDC/JPYC/PayPay等）
  - 金额（¥3,450 / 15.2 USDC）
  - 状态徽章（处理中/成功/失败）
  - 时间戳（刚刚 / 2分钟前）
- **交互**:
  - 点击查看详情
  - 自动滚动最新交易
  - 暂停/继续按钮
- **动画**: Motion (Framer Motion) 实现滑入动画

#### 收入趋势图表（使用 recharts）
```typescript
数据维度:
- X轴: 时间（小时/天/月，可切换）
- Y轴: 金额（¥）
- 双Y轴: 加密货币金额（USDC/JPYC）
- 图表类型: 面积图 + 折线图
- 可筛选: 支付方式、货币类型
```

#### 支付渠道分布（饼图）
- **USDC渠道**:
  - Binance Pay
  - OKX Pay
  - Bybit Pay
  - Coinbase Pay
  - 链上转账 (WalletConnect/MetaMask)
- **JPYC渠道**:
  - Polygon
  - Ethereum
  - Avalanche
- **日元渠道**:
  - Apple Pay
  - PayPay
  - Rakuten Pay
  - Credit Card

### 2. 交易记录 (Transactions)

#### 筛选工具栏
```
┌───────────────────────────────────────────────────────────┐
│ 🔍 搜索订单号/钱包地址                                     │
│                                                            │
│ [日期范围] [支付类型▼] [状态▼] [货币▼] [导出CSV]         │
│  今天         全部        全部    全部                      │
└───────────────────────────────────────────────────────────┘
```

**筛选选项**:
- **支付类型**: 加密货币 / 日元 / 全部
- **状态**: 成功 / 处理中 / 失败 / 已退款 / 全部
- **货币**: USDC / JPYC / JPY / 全部
- **支付渠道**: 所有具体渠道（下拉多选）
- **金额范围**: 自定义输入
- **日期范围**: 快捷选择（今天/昨天/本周/本月/自定义）

#### 交易列表表格（使用 ShadCN Table）
```
┌────────┬──────────┬────────────┬──────┬──────┬────────┬────────┐
│ 订单号  │ 时间      │ 支付方式    │ 金额  │ 状态  │ 网络    │ 操作   │
├────────┼──────────┼────────────┼──────┼──────┼────────┼────────┤
│ ORD-001│ 14:32:15 │ 💵 USDC    │¥3,450│ ✅   │ Base   │ [详情] │
│         │          │ Binance Pay│15.2  │成功  │        │        │
├────────┼──────────┼────────────┼──────┼──────┼────────┼────────┤
│ ORD-002│ 14:28:40 │ 💴 JPYC    │¥2,800│ 🔄   │Polygon │ [详情] │
│         │          │ MetaMask   │2,800 │处理中│        │        │
├────────┼──────────┼────────────┼──────┼──────┼────────┼────────┤
│ ORD-003│ 14:25:19 │ PayPay     │¥1,500│ ✅   │ -      │ [详情] │
│         │          │            │      │成功  │        │        │
└────────┴──────────┴────────────┴──────┴──────┴────────┴────────┘
```

**列字段**:
1. **订单号**: 可复制，点击查看详情
2. **时间**: 精确到秒，悬停显示完整时间戳
3. **支付方式**: 
   - 图标 + 货币类型
   - 次行显示具体渠道
4. **金额**: 
   - 主行显示日元金额
   - 次行显示加密货币金额（如适用）
5. **状态徽章**: 
   - ✅ 成功 (绿色)
   - 🔄 处理中 (蓝色，带动画)
   - ❌ 失败 (红色)
   - 🔙 已退款 (橙色)
6. **网络**: 显示区块链网络（仅加密货币）
7. **操作**: 查看详情按钮

**分页**:
- 每页显示 20/50/100 条（可选）
- ShadCN Pagination 组件
- 显示总数和当前页

## 关键用户流程

### 查看交易详情流程
1. 在交易记录页面点击"详情"按钮
2. 右侧滑出Sheet显示完整交易信息
3. 可复制交易哈希、钱包地址等
4. 可点击"在浏览器查看"跳转到区块链浏览器
5. 可执行退款操作（弹出确认对话框）

### 配置钱包地址流程
1. 进入"收款配置"页面
2. 选择对应网络卡片
3. 点击"编辑"按钮
4. 输入钱包地址
5. 系统验证地址格式
6. 保存并显示成功Toast
7. 显示"已验证"徽章

### 导出报表流程
1. 在交易记录页面设置筛选条件
2. 点击"导出CSV"按钮
3. 显示导出进度Toast
4. 下载包含筛选数据的CSV文件
